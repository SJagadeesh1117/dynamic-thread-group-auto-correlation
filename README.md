# Dynamic Thread Group Auto Correlation

This repository contains a single drop-in JMeter extension jar:

- `00_ApacheJMeter_enhanced_features.jar`

It bundles:

- Dynamic Load Thread Group
- Auto Correlation
- Startup result file popup fix

## Install

1. Stop JMeter.
2. Copy `00_ApacheJMeter_enhanced_features.jar` into `lib/ext/` of your JMeter installation.
3. Start JMeter.
